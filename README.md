## **Instalasi**

1. **Clone Repository**

   ```bash
   git clone https://github.com/username/repo-name.git
   ```

2. **Masuk ke direktori proyek**

   ```bash
   cd repo-name
   ```

3. **Install dependencies**

   ```bash
   npm install
   # atau jika menggunakan yarn
   yarn install
   ```

4. **Jalankan aplikasi**

   ```bash
   npm start
   # atau jika menggunakan yarn
   yarn start
   ```
